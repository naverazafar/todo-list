# Priority Task Manager

A Flask-based to-do app with priority sorting and visual indicators.

## Features
- Create, edit, and delete tasks
- Assign priority levels (High / Medium / Low)
- Automatic priority sorting
- Ribbon labels to visually indicate priority
- Backend with SQLAlchemy + SQLite

## Tech Stack
- Python
- Flask
- SQLAlchemy
- HTML / CSS
- Semantic UI

## Contributions beyond tutorial
- Added priority field and sorting (high to low) priority logic
- Ribbon labels for priority
- Minor CSS tweaks to improve layout

## Run Locally
1. Clone repo  
2. Install dependencies: `pip install -r requirements.txt`  
3. Create virtual environment: `python3 -m venv venv`  
4. Run app: `python app.py`
